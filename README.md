# ServerCV | Professional Discord Resumes

Build a verified portfolio of your server contributions. Perfect for staff applications and sharing your achievements in the Discord community. 

Check out the website at https://servercv.com.

## Features

- **Verified History**: Request endorsements from server owners or admins to verify your roles. Build credibility with confirmed data.
- **Instant Portfolio**: Generate a beautiful, shareable link (`/u/username`) that showcases your entire Discord career.
- **Endorsement System**: Server owners and admins can review, approve, or reject experience claims directly through the dashboard.
- **Premium Features**:
  - **Vanity URLs**: Custom profile links (e.g., `servercv.com/u/ian`).
  - **Unlimited Experiences**: Bypass the free tier limit.
  - **Pinned Experiences**: Highlight your most important roles.
  - **Extended Socials**: Add more social links to your profile.
  - **Server Vanity URLs**: Custom server links if you own them (e.g., `servercv.com/s/minecraft`).
- **Secure & Private**: Uses Discord OAuth2 for authentication. You control what is displayed.

## How It Works

1. **Connect**: Login with Discord to securely access your server history.
2. **Verify**: Request endorsement from server owners or admins. Once approved, your role is added to your verified resume.
3. **Showcase**: Share your unique profile link with server owners and applications.

## Tech Stack

- **Backend**: Python, Flask
- **Database**: Firebase Realtime Database
- **Authentication**: Discord OAuth2
- **Frontend**: HTML5, Tailwind CSS (via CDN), JavaScript
- **Payments**: PayPal Integration
